Kinopoisk Reviews Sentiment Analysis

Задача проекта состоит в предположении окраски новой рецензии на основе её текста - является ли рецензия положтельной, отрицательной или нейтральной. 

В проекте используется набор данных по ссылке ниже, состоящий из 131 583 рецензии с сайта Кинопоиск, из которых 87 101 положительная рецензия, 19 804 отрицательные рецензии и 24 678 нейтральные рецензии:

https://www.kaggle.com/datasets/mikhailklemin/kinopoisks-movies-reviews/data

Модель тренировалась на 80% этих данных и тестировалась на оставшихся 20%.

В файле data processing файлы из набора данных сливаются в один файл all_reviews.csv с сохранением информации об окраске рецензии в отдельном столбце.

В файле training and testing данные разделяются на тренировочную и тестовую выборки, модель тренируется, сохраняется в файл sentiment_model.bin, после предсказания на тестовой выборке производится оценка качества модели.

В файле testing 2 дополнительно представлена возможность определить окраску рецензии не из набора данных, а введённой с клавиатуры.
